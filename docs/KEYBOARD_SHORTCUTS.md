## 🔧 Technical Details

### **How It Works**
- Shortcuts are **global** - work from anywhere
- POS shortcuts only work in **Point of Sale** screen
- Shortcuts **don't trigger** when typing in input fields
- Uses browser's native keyboard events
- No conflicts with OS shortcuts

### **Keyboard Focus**
```
Normal State:
- Shortcuts active
- Function keys work
- Ctrl+Number works

Input Field Focus:
- Shortcuts PAUSED
- Type normally
- Press Esc or Tab to exit field
- Shortcuts active again
```

## 🐛 Troubleshooting

### **Shortcut Not Working?**
1. Check if you're in an input field (Tab out)
2. Ensure Caps Lock is OFF
3. Try pressing F1 to test shortcuts
4. Refresh page (F5) if issues persist

### **Wrong Action Triggered?**
- Check if Ctrl/Shift/Alt is stuck
- Restart application
- Check keyboard hardware

### **Help Modal Won't Close?**
- Press Esc key
- Click X button at top
- Click "Got it!" button
